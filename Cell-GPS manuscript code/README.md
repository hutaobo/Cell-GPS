# Cell-GPS manuscript code

This folder reorganizes the located Cell-GPS Science manuscript analysis code by manuscript result.

The notebooks are written in English and are intentionally output-free. They preserve the original data paths found during the code inventory and add detailed comments explaining each computational step. Some notebooks require data on the Y drive or on the A100 server paths used during manuscript preparation.

## Organization

- `main_figures/`: one notebook per main figure with located code.
- `supplementary_figures/`: one notebook per supplementary figure with located code.
- `supplementary_tables/`: one notebook per supplementary table with located code.

If no reliable code was found for a panel, no separate panel notebook is included. Final multi-panel assembly was usually done from generated PDFs/PNGs, so these notebooks focus on the computational source for each figure/table rather than Illustrator/PowerPoint assembly.

## Provenance

The mapping is based on `docs/cellgps_science_manuscript_code_inventory.md` plus direct inspection of the current worktree, the Y drive, and A100/PDC search results.
